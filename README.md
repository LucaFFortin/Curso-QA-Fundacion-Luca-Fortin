# Curso QA - Fundación Luca Fortín

Este repositorio contiene materiales, ejercicios y documentación relacionados con el curso de **Quality Assurance (QA)**, desarrollado en colaboración con la Fundación y la Universidad Tecnológica Nacional (UTN).  
El objetivo es brindar una base sólida en **testing manual y automatizado**, buenas prácticas de QA y metodologías ágiles aplicadas al desarrollo de software.

---

## 📚 Contenido

- **Ejercicios prácticos**: casos de prueba, ejecución de tests automatizados.
- **Proyectos de práctica**: ejemplos de aplicación en entornos simulados (saucedemo.com).

---

## 🚀 Objetivos del curso

- Comprender el rol del QA en el ciclo de vida del software.
- Aprender a diseñar y ejecutar casos de prueba efectivos.
- Introducirse en **automatización de pruebas** y buenas prácticas de desarrollo.
- Desarrollar habilidades de trabajo en equipo bajo metodologías ágiles.

---
